# Podcasts Manager

### Descrição
Um app ao estilo spotify onde eu possa claasificar diferentes episódios de podcasts por categoria..

### Domínio
Podcasts feitos em video

### Features
- listar os podcasts em seções de categorias
  - [saúde, fitness, inglês, programação, etc]
- Filtrar episódios por nome de podcasts


## Como fazer
  Retornar em uma API os seguintes itens: editor do podcast, título do podcast, imagem de capa, link, caterogia..

```js
[
    {
    podcastEditor: "Flow Podcast",
    podcastTitle: "MIGUEL NICOLELIS - Flow #445",
    videoId: "fkVlpbDmpEU",
    coverImg: "https://i.ytimg.com/vi/fkVlpbDmpEU/maxresdefault.jpg",
    link: "https://www.youtube.com/watch?v=fkVlpbDmpEU&list=PLWieWKZeFoVSyI--JwyTth19kUrMrF5Am&index=3",
    categories: ["ciência", "tecnologia"]
```
